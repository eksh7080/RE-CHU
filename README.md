### 개발자 이력서 토론 플랫폼을 제공합니다.

1. 커뮤니티에 이력서를 첨부하여 다른 사람들과 토론할 수 있습니다.
2. 활동량에 따라 포인트를 지급하는 등 게이미피케이션적인 요소를 가미합니다.
3. 원하는 유저에게 이력서 첨삭을 요청 후 채팅을 통해 피드백을 받을 수 있습니다.
